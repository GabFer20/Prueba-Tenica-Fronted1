# Gestor de Activos Corporativos - React + Tailwind (Plantilla)

Proyecto creado según la prueba técnica. Contiene:
- React + Vite
- TailwindCSS
- json-server (API falsa) con `db.json`
- Rutas públicas y privadas, autenticación simulada con localStorage
- CRUD de activos, vista de empleados, panel con estadísticas

## Requisitos
- Node >= 18
- npm

## Instalación rápida
1. `npm install`
2. `npm run serve:api` (levanta json-server en http://localhost:3001)
3. `npm run dev` (levanta Vite en http://localhost:5173)
